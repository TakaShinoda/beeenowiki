# beeenowiki
[![CircleCI](https://circleci.com/gh/TakaShinoda/beeenowiki/tree/master.svg?style=svg)](https://circleci.com/gh/TakaShinoda/beeenowiki/tree/master)

## Get Start

```
git clone https://github.com/TakaShinoda/beeenowiki
yarn
```

## Develop

### Build
```
yarn build
```

or

```
yarn dev
```

### Deploy
```
yarn deploy
```
